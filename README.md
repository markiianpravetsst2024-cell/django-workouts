# 🏋️ Workout Tracker 



У проекті реалізовано повний цикл **CRUD**:
- **Create**: Створення нового тренування через форму (`ModelForm`).
- **Read**: Перегляд списку всіх тренувань та детальної інформації про кожне.
- **Update**: Можливість редагування існуючих записів та швидка кнопка "Виконано".
- **Delete**: Безпечне видалення записів з підтвердженням.


## Опис моделей 
- **Python 3.14**
- **Django 6.0.4**
- **SQLite** 


.

##  Як запустити локально

1. **Клонуйте репозиторій або скопіюйте файли.**
2. **Створіть та активуйте віртуальне середовище:**
   ```bash
   python -m venv .venv
   source .venv/Scripts/activate  # Для Windows
3. **Встановіть Django:**
   ```bash
    pip install django
4. **Застосуйте міграції:**
    ```bash
   python manage.py migrate
5. **Запустіть сервер:**
    ```bash
   python manage.py runserver
